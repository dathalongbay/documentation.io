Thuộc tính flex-grow

Thuộc tính này sẽ chỉ ra flex item sẽ tăng kích thước chiều rộng lên tương ứng với các flex item còn lại

Giá trị này là 1 số, mặc định là 0

Ví dụ flex-grow: 0 ( mặc định )

Ví dụ flex-grow: 1 ( giãn ra 1 đơn vị )

Ví dụ flex-grow: 2 ( giãn ra 2 đơn vị )

Ví dụ flex-grow: 3 ( giãn ra 3 đơn vị )

Flex grow mặc định flex-grow = 0

Code CSS

cột 1 flex-grow: 0
cột 2 flex-grow: 0
cột 3 flex-grow: 0

Sau khi sử dụng thuộc tính css flex-grow > 0

cột 1 flex-grow: 1
cột 2 flex-grow: 0
cột 3 flex-grow: 9

Giải thích :
thuộc tính flex-grow sẽ giãn ra để chiếm toàn bộ diện tích của flex container nếu flex container còn khoảng trống
flex container ở ví dụ trên rộng 600px mỗi thẻ div bên trong rộng 100px.
flex-grow mặc định là 0 có nghĩa là sẽ không giãn ra .
thẻ mẹ có độ rộng là 600px . mỗi thẻ con có độ rộng là 100px .
có 3 thẻ div bên trong flex container . nên độ rộng đã chiếm là 100*3 = 300px
khoảng trống còn thừa là 600px-300px = 300px
ta đặt cột 1 giãn 1 đơn vị . cột 2 không giãn . cột 3 giãn 9 đơn vị
tổng là sẽ giãn 10 đơn vị . ví dụ này lấy tổng đơn vị là 10 cho các bạn dễ hiểu
khoảng trống là 300px
vậy mỗi đơn vị giãn tương ứng với 300px/10 = 30px
cột 1 sẽ tăng từ 100px + (30px*1) giãn lên 130px
cột 2 không tăng do flex-grow là 0 : 100px + (30px*1) = 100px
cột 3 sẽ tăng từ 300px lên (100px + (9*30px)) = 370px
Tổng của flex-grow là các flex-grow của các flex item nằm trong flex container Tổng flex-grow = (Cột 1 flex-grow là 1) + (Cột 2 flex-grow là 0) + (Cột 3 flex-grow là 9) = 10 Nên nhớ mỗi đơn vị flex-grow khi quy đổi ra px là 1 sự tương đối tùy vào từng trường hợp

Ví dụ 1 :

cột 1 flex-grow: 2
cột 2 flex-grow: 0
cột 3 flex-grow: 6

Tổng chiều rộng flex container : 600px
Kích cỡ mặc định của 1 thẻ : 100px
Kích cỡ 3 thẻ mặc định đã chiếm : 100*3 = 300px
Khoảng trống còn thừa : 600-300 = 300px
Tổng flex-grow của các cột : 2 + 0 + 6 = 8
Mỗi đơn vị flex-grow quy đổi ra chiều rộng : 300px/8 = 37.5
Độ rộng cột 1 : width mặc định + width flex-grow tăng = width thực tế
Độ rộng cột 1 : 100 + (37.5*2) = 175px
Độ rộng cột 2 : width mặc định + width flex-grow tăng = width thực tế
Độ rộng cột 2 : 100 + (37.5*0) = 100px
Độ rộng cột 3 : width mặc định + width flex-grow tăng = width thực tế
Độ rộng cột 3 : 100 + (37.5*6) = 325px
